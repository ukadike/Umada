# Changelog

This repository keeps its change record as an append-only archive-recovery log rather
than a conventional version-bumped changelog, since it tracks *what was ingested from
where*, not software releases: see
[`15_archive_recovery/README.md`](15_archive_recovery/README.md). Entries are added,
never rewritten — corrections are logged as new entries, not silent edits to old ones.

For visual-artifact-specific ingestion history, see
[`20_visual_canon_registry/README.md`](20_visual_canon_registry/README.md).
