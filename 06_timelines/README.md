# Timelines

The canonical 17-era timeline (Era 0 — Ancient Memory, through Era 16 — Umada) lives in
`01_canon/CIVILIZATIONAL_LEDGER.md`, with the structured form in `data/timeline.json` and
`data/civilizational_ledger.json`. This directory is intentionally kept free of a
duplicate copy so the timeline has one source of truth; it holds derived timeline
materials (e.g. sub-era detail, cross-references) once fragments justify adding them.
