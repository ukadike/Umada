# UMADA

> This is not a world that was built. It is a world being remembered.
> Some records survived. Some did not. Some stories were erased. Some refused to disappear.
> If you stay long enough, you may help us remember.

UMADA is a canon-governed world, told as a recovered archive rather than a finished script.
Every fact you find here carries a status. Every gap is marked, not papered over. Nothing
is invented to fill a silence — if something isn't known yet, the archive says so and
waits.

This file is the welcome mat for visitors and fans. If you're looking for the
contributor/build-process README instead, see [`README.md`](README.md).

## Canon status, at a glance

Every fact in this archive is tagged so you always know how solid the ground is:

| Status | Meaning |
| --- | --- |
| **LOCKED** | Confirmed. Won't drift or get quietly swapped for a variant. |
| **PROBABLE** | Strongly indicated by the source material, not yet fully confirmed. |
| **EMERGING** | Named and directional — detail is still arriving. |
| **OPEN QUESTION** | Genuinely unresolved. Tracked openly, never guessed at. |
| **RETIRED** | A past error or dropped variant, kept on record so it can't sneak back in. |
| **AWAITING FRAGMENT** | Named somewhere, but the actual material hasn't arrived yet. The index entry exists; its contents don't, until it does. |

Where a realized image plate or the animatic disagrees with anything written, the plate or
animatic wins — see [Visual Canon](sections/visual-canon.html).

## Accessibility is infrastructure here, not an add-on

Every Nago sign, image, and the animatic carry alt text, plain-language summaries, and
(where they exist yet) captions and tactile-form descriptions. Nothing in this archive
relies on color alone, hover alone, or perfect vision or hearing to be understood. See
[Accessibility](sections/accessibility.html) for the full baseline.

## How to contribute

Fans contribute **signal, not automatic canon**. Whatever you send in passes through the
Faraday Box, which receives audience signal, protects canon, filters harm, preserves your
traces, and returns prompts — it never lets a fan suggestion overwrite the locked record
on its own.

A few prompts the Faraday Box asks visitors:

- What sign would your community create?
- What does care look like after collapse?
- What memory should never be erased?
- What would your relay mark be?

Visit the [Faraday Box](sections/faraday-box.html) to take part.
