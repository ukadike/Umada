# Memory Tablets and Memory Stones

Source: `UMADA_CANON_INDEX.md` and `umada_book_one_claude_package_v3.zip`
(`estate-of-memory/memory-tablets/nago_memory_tablets.md`), both delivered by
Kemi 2026-06-25. See `13_fragments/2026-06-25_canon_index_directive.md` and
`13_fragments/2026-06-25_book_one_v3_narrative.md` for raw provenance.

## Memory Tablets

Memory Tablets preserve Nago as a living archive. They are not read only
left-to-right; they are read holistically.

**Registers:**

- Central register: principal concepts.
- Surrounding text: context, commentary, history, song, or law.
- Bottom row: glyph reference, teaching row, or dictionary line.

**Educational use:** Children first learn core glyphs. Later they learn
combinations, oral traditions, tactile readings, sign equivalents, and
historical commentaries.

**Canonical interpretation:** Memory Tablets are not only storage. They are
civic pedagogy. They teach a population how to remember together.

This register structure is EMERGING and single-fragment-sourced; it has not
been checked against the realized plates in `assets/img/`.

## Memory Stones

Examples named in the Canon Index: Quartz, Coral, regional stones, Dead
Glass. Memory Stones can be activated or coded to receive Ether Power.

**Disambiguation note:** the material "Quartz" listed here is not stated to
be related to the character **Quartz**, Ada's survivor-unit companion
("Quartz carries"). No fragment links them. Treat as two unrelated entities
unless a future fragment states otherwise.

## Ether Power

A free-energy-like system accessed through activated/coded Memory Stones and
Dead Glass. Per the Canon Index's explicit rule, Ether Power is **not**
biological-interface based.
