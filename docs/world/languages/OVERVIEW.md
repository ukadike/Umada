# Language Overview

Status: EMERGING / HIGH PRIORITY. The only documented language/signage system in UMADA
is **Nago**. Full detail: `07_languages/NAGO_SYMBOLOGY_BIBLE.md`,
`07_languages/NAGO_SIGNAGE_GUIDE.md`, `07_languages/NAGO_GLYPH_SYSTEM.md`. The realized
plates (`assets/img/PLATE_NAGO_SYMBOLOGY_OF_UMADA.png`,
`assets/img/PLATE_PROTO_NAGO.png`) are the visual source of truth.

## What Nago is

Nago is the visual operating system of Umada: survival vocabulary before grammar,
exchanged in markets, bread stops, rescue routes, and mixed human/chimera communities,
later becoming civic, accessibility, memory, and relay infrastructure. It carries a
real-world provenance note — it draws on Nagô / Yoruba heritage — and is, within UMADA,
a fictional, evolving sign system, not a depiction of any real sign language or sacred
script.

## Two axes (never merged)

- **Function families (signage, seven):** Survival, Care, Signal, Movement, Civic,
  Memory (a rename of "Witness," not a deletion), Environment (new, EMERGING, no signs
  seeded yet).
- **Motif families (narrative, five):** Wound, Relay, Archive, Storm, Visitor.

## Core survival signs (locked, eight)

Bread, Water, Safe, Danger, Carry, Home, Help, Rest — locked to the realized plate.
Witness and Relay are canonical Nago signs but belong to the Memory and Signal families,
not the core-survival set.

## Multimodal by default

Every Nago sign exists as gesture, flat visual mark, raised tactile mark, verbal
description, screen-reader label, public signage icon, map symbol, and episode graphic —
per `00_governance/ACCESSIBILITY_BASELINE.md`. No sign relies on color alone, hover
alone, perfect vision alone, literacy alone, or hearing alone.
