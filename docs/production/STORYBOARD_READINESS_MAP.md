## Episode-level block (applies regardless of per-anchor readiness)

Per the root `README.md` and `docs/production/ROADMAP.md`, Episode 1 / the Cape
Wipeout opening sequence has been deliberately left unwritten. Several anchors above
are individually "Ready," but that does not authorize storyboarding them — episode
authorization is a separate, higher-level gate than anchor readiness. This map exists
so that *when* that gate opens, the ready anchors are already identified.

## Relationship to the realized rescue animatic

The rescue animatic (`assets/video/UMADA_Friedmandostorp_Rescue_Animatic.mp4`) already
realizes several of the anchors above (Ada injured, Tallai finding her, Q and Quartz
present, ADA-1's body-over-record logic) as a finished artifact predating this map. Any
future storyboard covering the same beats should treat the animatic's recovered caption
transcript (`01_canon/VISUAL_CANON_REGISTRY.md`) as the visual source of truth for that
material, per `00_governance/VISUAL_LAYER_RULES.md`'s plates-win rule.
