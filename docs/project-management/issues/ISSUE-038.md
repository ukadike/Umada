# ISSUE-038: Build an asset-naming and versioning convention for animation files

**Category:** Animation Preparation

## Objective

Define a naming/versioning scheme for future animation source and export files so `assets/` doesn't accumulate ad hoc names once production starts.

## Tasks

- Propose a convention.
- Confirm the convention with Kemi.
- Document it in `docs/production/ROADMAP.md`.

## Acceptance Criteria

- The convention is documented before any animation file is added to the repository.

## Dependencies

- docs/production/ROADMAP.md
