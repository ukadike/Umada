# ISSUE-025: Plain-language summary pass for archival documents

**Category:** Accessibility

## Objective

Add plain-language summaries alongside dense archival text where the baseline requires them but none yet exist.

## Tasks

- Identify which canon-engine documents are dense/archival (e.g. `01_canon/FOUNDATIONAL_STRATA.md`, `CIVILIZATIONAL_LEDGER.md`).
- Check whether a plain-language summary already exists on the corresponding site section.
- Add one where missing, citing the source rather than paraphrasing facts differently.

## Acceptance Criteria

- Every public-facing section drawing on a dense canon document has a plain-language summary.
