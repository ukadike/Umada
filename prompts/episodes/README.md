# Episode Prompts

Reserved. Shot- or scene-level image-generation prompts depend on a storyboard, which
depends on a script — and Episode 1 has deliberately not been scripted yet
(`11_episodes/README.md`, `docs/production/ROADMAP.md`). Writing episode-specific
prompts before that exists would mean inventing scenes, which this repository does not
do.

The one exception already in this repository predates any script: the five locked Ada
images and the one locked Tallai/Ada image were generated from already-locked *character*
canon, not from episode scenes — see `prompts/characters/`. That precedent doesn't
extend to inventing new scenes.

See `11_episodes/EPISODE_01_CAPE_WIPEOUT.md` for the locked facts a future script and
its prompts will need to be consistent with.
