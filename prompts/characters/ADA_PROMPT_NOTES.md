# Ada — Prompt Notes

Status: **Locked reference exists** (five images). These notes restate what's already
approved, for generating *additional, consistent* Ada imagery — they do not add anything
beyond `02_characters/ADA_DOSSIER.md` and `00_governance/CANON_STATUS.md`.

## Locked visual elements (reuse in any new prompt)

- Nigerian woman, slim build.
- Dark hair, default look slicked back into a ponytail.
- Thin silver necklace with a small, solid heart pendant (not a locket) — gift from
  Kassey/KC.
- Woodland-pattern camo military uniform; name tape reading "ADA"; gold collar rank
  insignia ("medium brass," read as mid-ranking officer); wings/pilot-style chest badge
  (aviation background vs. decorative — open, `01_canon/OPEN_QUESTIONS.md` #31); sleeve
  flag patch (design unspecified, but a "NIGERIA" sleeve patch is confirmed legible in
  the wasteland reference); web belt with holstered sidearm and knife; wristwatch.
- Demeanor under pressure: cold and methodical, not loud.
