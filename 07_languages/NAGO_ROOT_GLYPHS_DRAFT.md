## Premise

Nago is not an alphabet. It is a semantic writing and communication system.
Each glyph represents a core concept rather than a sound. Meaning emerges
through relationship, sequence, gesture, position, touch, song, and context.

## A third, separate layer

This semantic root-glyph layer is distinct from, and must not be merged
with, the repo's two existing Nago axes:

- **Function families (signage):** Survival, Care, Signal, Movement, Civic,
  Memory, Environment.
- **Motif families (narrative):** Wound, Relay, Archive, Storm, Visitor.

The root glyphs below describe semantic concepts a glyph can carry; they do
not replace, rename, or reclassify any locked function-family or
motif-family sign (e.g. the locked core survival signs: Bread, Water, Safe,
Danger, Carry, Home, Help, Rest).

## Equal modalities

Nago exists as: visual glyphs, tactile glyphs, Hand Relay / sign language,
spoken language, sung forms, carved tablets, memory stones, and Dead Glass
inscriptions. No single modality is the real Nago. They are all Nago.

## Canonical Root Glyphs v0.1

| Glyph Concept | Meaning |
|---|---|
| Circle over crescent | Community / Home |
| Three horizontal bars / staff form | Governance / Law |
| Concentric target | Memory |
| Eye | Observation |
| Spiral diamond | Knowledge / Inquiry |
| Dotted matrix | People / Community |
| Branch | Growth / Life |
| Vertical staff | Stewardship |
| Divided circle | Balance |

| Glyph Combination | Interpretation |
|---|---|

| Community + Stewardship | Governance |
| Growth + Memory | Learn |
| Stewardship + Balance | Justice |

## Design principle

A language that cannot be accessed by all citizens cannot govern all
citizens.
