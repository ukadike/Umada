# Nago Symbology Bible v1 (consolidated)
## The Visual Operating System of Umada

Status: EMERGING / HIGH PRIORITY

## Provenance (real-world)

"Nago" draws on Nagô / Yoruba heritage. Within UMADA it is a fictional, evolving
survival-and-care sign system — not a depiction of any real sign language or sacred
script. Treat every rooted term with provenance and care. Fan and collaborator
contributions inherit this commitment. This note is separate from the in-world
HONOR & ORIGIN account below.

## Core Thesis

Nago begins before grammar. Need comes first. The earliest known signs are survival
signs, exchanged in markets, bread stops, rescue routes, damaged streets, and mixed
human/chimera communities.

Nago later becomes: civic infrastructure, accessibility infrastructure, memory
infrastructure, relay infrastructure, public signage, tactile navigation, fan
participation language, and the visual identity for Umada.

Nago is not generic fantasy writing. Nago is a system of care, warning, witness, and orientation.

# Core Survival Signs (canonical eight)

Locked to the realized plate. The canonical core set is:

**Bread · Water · Safe · Danger · Carry · Home · Help · Rest**

(Witness and Relay remain canonical Nago signs but belong to the Memory and Signal
families respectively; they are not part of the core-survival set.)

## Bread Sign

Status: LOCKED / FOUNDATIONAL
Gesture: Fingers to mouth. Palm open.
Meaning: I am hungry. I ask for bread. Not begging. Survival with dignity.
Context: Friedmandostorp bread stop. Early Nago market sign. Tallai / Talla known at the bread stop.

# Symbol Families (function axis — seven)

These organize signage by purpose. Note: the earlier "Witness" family is now **Memory**
(rename, not deletion); **Environment** is new.

## 1. Survival — to live
Seed signs: Bread, Water, Shelter, Safe, Danger, Carry, Home.

## 2. Care — to heal and protect
Seed signs: Heal, Feed, Rest, Hold, Protect, Stabilize, Do not leave, Help.

## 3. Signal — to listen and connect
Seed signs: Listen, Relay, Tower, Signal, Storm, Containment, Return signal.

## 4. Movement — to travel and return
Seed signs: Go, Return, Gather, Cross, Carry, Follow, Wait.

## 5. Civic — to build together
Seed signs: Council, Shared resource, Archive, School, Clinic, Water point, Safe route, Public witness.

## 6. Memory — to remember and learn (formerly "Witness")
Seed signs: Witness, Record, Remember, Hidden record, False record, Body before record.

## 7. Environment — to honor the world
Seed signs: AWAITING design (status EMERGING). Seed with care-of-place / land / water / weather marks.

# Visual Grammar (from the plate)

Built on the Golden Rectangle and ancient geometry — for legibility and memory, not mystification.

Primitives and meanings:
- Circle — memory / continuity / shared memory
- Open Circle — open access / shared
- Broken Circle — lost / interrupted memory / broken record
- Three Rings — relay / signal
- Hand + Ring — witness / presence

---

# Tactile System

Every Nago sign must exist as: gesture, flat visual mark, raised tactile mark, verbal
description, screen-reader label, public signage icon, map symbol, episode graphic.

Tactile rules: avoid thin details; clear edges; high-contrast relief; support fingertip
scanning; redundant text labels; never rely on color alone; design for low vision and
blind users from the beginning.

---

# Accessibility Requirements

Every Nago symbol entry must include: alt text, plain-language meaning, tactile
description, context of use, risk of confusion, cultural/provenance note, and whether it
can be used by humans, chimeras, cyborgs, or all.
