# Languages

Status: EMERGING

The Nago language and signage system — Umada's visual operating system: survival
language before grammar, accessibility infrastructure, tactile signage, civic memory,
relay/signal system, market-originated communication, later civilizational symbol
system. See `CLAUDE.md` for the two axes (function families vs. motif families) that
must not be merged.

- **[`NAGO_SYMBOLOGY_BIBLE.md`](NAGO_SYMBOLOGY_BIBLE.md)** — the full symbology
  reference: provenance note (Nagô/Yoruba), function families, motif families, and the
  canonical core survival signs.
- **[`NAGO_GLYPH_SYSTEM.md`](NAGO_GLYPH_SYSTEM.md)** — the glyph system structure.
- **[`NAGO_ROOT_GLYPHS_DRAFT.md`](NAGO_ROOT_GLYPHS_DRAFT.md)** — draft root glyphs,
  not yet locked.
- **[`NAGO_SIGNAGE_GUIDE.md`](NAGO_SIGNAGE_GUIDE.md)** — sign-type and public-sign
  template detail; the accessibility-facing companion to the symbology bible.

The realized plates in `assets/img/` (`PLATE_NAGO_SYMBOLOGY_OF_UMADA.png`,
`PLATE_PROTO_NAGO.png`) are the visual source of truth — where any document here
disagrees with a plate, the plate wins. Structured sign data lives in
`data/nago_signs.json`, `data/nago_proto_signs.json`, `data/nago_root_glyphs.json`, and
`data/nago_symbology.json`, and is rendered live on `sections/nago-symbology.html` and
`sections/language-signage.html`.
