`main.css` — the presentation layer. Implements the monochrome graphite palette and type
system locked in `00_governance/VISUAL_LAYER_RULES.md` (Helvetica Neue / Georgia / Courier
Prime; paper/ink/line/muted custom properties), plus the `.status-tag` component, which
distinguishes canon status by border style rather than color alone, per
`00_governance/ACCESSIBILITY_BASELINE.md`. Diegetic images and video stay in the world's
own color; only the surrounding chrome is monochrome.
