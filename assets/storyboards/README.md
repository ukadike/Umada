## Shot/file naming convention (PROBABLE — pending Kemi's confirmation)

Per ISSUE-034:

- **Authorized episode storyboards:** `EP##-SC##-SHOT##` (episode, scene, shot number —
  e.g. `EP01-SC01-SHOT04`). Used once an episode's storyboarding is actually authorized.
- **Pre-authorization proof-of-concept / style-test shots** (e.g. exploratory AI-image
  prompt tests not yet tied to an authorized episode storyboard): prefix with the
  sequence short name instead of an episode number — e.g. `CW-PROOF-001` — so
  proof-of-concept work is never mistaken for authorized episode content. This
  reconciles the `CW-001`-style prefix used by the 2026-06-27 studio-pipeline fragment
  (`13_fragments/2026-06-27_studio_pipeline_package.md`) with the episode-scoped
  convention below, without letting unauthorized content silently look canonical.

This convention is PROBABLE pending Kemi's confirmation (see Tiny Question in the
2026-06-27 fragment) — not yet exercised on any real file, since no storyboard content
has been authorized.
