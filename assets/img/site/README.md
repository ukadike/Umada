# Site-presentation assets

This folder holds images **derived from** the realized plates in `assets/img/` for use in
the README and similar presentation contexts — crops, composites, banners. It is not part
of the visual canon and adds no new facts.

- `README_HERO_BANNER.png` — a golden-ratio (1.618:1) crop of the top portion of
  `assets/img/PLATE_NAGO_SYMBOLOGY_OF_UMADA.png` (header, design principles, the Visual
  Grammar / Golden Rectangle panel, and the eight core survival signs), used as the README
  hero image. The source plate remains the visual source of truth; if they ever disagree,
  the source plate wins.
